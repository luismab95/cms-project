Hello,

After cloning all repository you must do the following activities

    1.- Extract a docker.tar.gz file    
            in linux run `tar -xzvf docker.tar.gz`

    2.- Create a docker container with docker files extracted.
            - postgres
            - redis
            - mongo

    3.- Install dependences in each repository.

    4.- Configure a API GATEWAY for MS repository
            - Use express gateway or any that you prefer

    5.- Configure all .env files in all repository.

    6.- Copy uploads folder to cms-ms-file in root dir   

    7.- Start all repository
            - for MS run `npm run start:dev`
            - for FRONT run `npm run start`

    8.- Login in Frontend with role Admin (review credentials_users.txt file).

    9.- Update Parameters in Frontend (Email credentials) and more.   

This is all enjoy =)
